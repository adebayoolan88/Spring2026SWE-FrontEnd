# UI Baseline Screenshots

This folder is the tracked location for migration baseline screenshots.

## Target screens
- Main catalog
- Auth modal
- Cart panel
- Product details modal
- Admin dashboard tables/forms
- Checkout success
- Checkout cancel
- Order details

## Required viewport variants
- Desktop width
- Mobile width

## State checklist
Use the checklist in `main/docs/ui-baseline/ui-state-checklist.md` during every migration batch and ensure screenshots for all available states are refreshed.
